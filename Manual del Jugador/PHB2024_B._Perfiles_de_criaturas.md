# Apéndice B: Perfiles de criaturas
Este apéndice contiene los perfiles de las criaturas mencionadas en el libro, principalmente en los capítulos de las clases, el equipo y los conjuros. Consulta el glosario de reglas para saber cómo se lee el perfil de una criatura y el Manual de monstruos para ver muchas más criaturas.

Los siguientes perfiles se presentan en orden alfabético. Cuando tu Dungeon Master use un perfil, es posible que cambie algunos detalles.
